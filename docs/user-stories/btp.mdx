# SAP Business Technology Platform (BTP)

The pages in the SAP BTP section will walk you through a complete end-to-end use case for the Crossplane provider for SAP BTP.

To realize the use case, you'll:

- [Install](/docs/crossplane-provider-btp/docs/user-stories/setup/install-provider-btp) and [configure](/docs/crossplane-provider-btp/docs/user-stories/setup/configure-provider-btp) the BTP provider.
- [Create and set up an orchestrated subaccount](/docs/crossplane-provider-btp/docs/user-stories/account/subaccount).
- [Configure the user access to you orchestrated subaccount](/docs/crossplane-provider-btp/docs/user-stories/account/usermanagement).
- [Create an instance of the Message Queuing service and subscribe to Business Application Studio](/docs/crossplane-provider-btp/docs/user-stories/services/create-services).
- Create a message queue and add two queue topics (coming soon).

:::image-light
![Big picture](/img/overview-orchestrate-service-all.png)
:::

Applying these steps manually would cost you  around 30 minutes of manual work for each environment you need.
Most enterprise projects run in numerous development stages and multiple regions.
Without automation, this process not only consumes significant manual effort but also increases the risk of errors.
